Este é um programa em Python desenvolvido para gerenciar e analisar o consumo de combustível e a eficiência de motoristas em viagens corporativas. O script calcula médias de consumo, custos individuais e identifica o motorista com melhor desempenho.

 Funcionalidades

- Cálculo de Médias Automatizado:** Calcula Km/L, Km por viagem e Litros por viagem.
- Análise de Custos:** Baseado no preço atual do combustível (R$ 5,80), calcula o investimento total por motorista e da empresa.
- Alertas de Desempenho:** Identifica automaticamente rendimentos abaixo do esperado (ex: consumo alto por viagem).
- Destaque de Eficiência:** Identifica o motorista que obteve a melhor média de Km/L.

 Tecnologias Utilizadas

* Python 3.x**: Linguagem principal.
* Lógica de Programação**: Listas, Loops, Fatiamento de dados (Slicing) e Condicionais.
